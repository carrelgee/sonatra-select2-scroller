Getting Started
===============

Prerequisites
-------------

This library require:

- jquery
- select2
- sonatra-jquery-scroller

Installation
------------

Installation is a quick:

Add library in your `bower.json`:

```json
{
    "dependencies": {
        "sonatra-select2-scroller": "latest"
    }
}
```

Launch bower:

```bash
$ bower install
```

Dev installation
----------------

### Use NPM

```
npm install
```

### Use Bower

```
bower install
```

### Use grunt

```
grunt serve
```

For ajax example, you must run this command in parallel:

```
grunt nodemon
```
